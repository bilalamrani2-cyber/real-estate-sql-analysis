# Real Estate SQL Analysis

This project is a beginner SQL database analysis project focused on real estate data.

## Project Overview
The database is designed to manage:
- Owners
- Houses
- Sales

The project includes SQL tasks such as:
- Creating tables with primary and foreign keys
- Inserting sample data
- Filtering and sorting records
- Using aggregate functions
- Grouping results
- Writing JOIN queries
- Analyzing sold and unsold houses

## SQL Concepts Used
- SELECT
- WHERE
- ORDER BY
- COUNT
- AVG
- MIN / MAX
- GROUP BY
- HAVING
- INNER JOIN
- LEFT JOIN
- ALTER TABLE

## Example Analysis Questions
- Which houses are not sold?
- What is the average price by house type?
- Which owner has the most houses?
- Which sales are above 100000?

## Tools
- MySQL
- SQL

## Purpose
This project was created to practice SQL database design and data analysis concepts for entry-level data analyst roles.
